# Admin Spec

## Implemented modules
- Dashboard metrics
- Product list
- Category list
- Order list
- Customer list
- Content editor shell (hero EN/VI fields)
- Media library placeholder
- Store settings form

## Behavior
- Admin runs inside same Next.js app under `/{locale}/admin/*`.
- Shared localization system with storefront.
- API routes are colocated in Next.js for one-deployment architecture.
